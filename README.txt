Snap.py: Programme simple permettant de prendre 5 photos d'une zone précise et d'en extraire les visages présents.
Filter.py: Programme permettant de filtrer les visages présents sur une photo afin de les isoler et de zoomer dessus afin de pouvoir 
y réaliser un algorithme de reconnaissance Facial

Pré requis: Créer un dossier Filtered (vide) et un dossier Unfiltered (vide lui aussi), cela permettra au programme de filter.py d'opérer sans 
problèmes.

Il vous faudra également les librairies python time et numpy pour snap.py
ET les librairies python glob, cv2, os pour Filter.py

lancez le programme, faites en sortes de présenter un visage devant la caméra, une fois réaliser vos photos doivent apparaître dans le dossier
Unfiltered.py

Lancez le programme Filter.py, les images filtrer apparaîtront dans le dossier Filtered.py
